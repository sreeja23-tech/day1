What is Browser?

A web browser is a software application that lets you view and interact on the World Wide Web

Example:-

Google Chrome, Safari and Microsoft Edge

What is Internet?

The internet is a global network of interconnected computers and devices that allows user to communicate and share information.

Example:-

Using the internet is browsing a website like google or sending an email.

What is URL?

Uniform Resource Locator is the web address that directs user to specific resource on the internet

Example:-

https://www.10000coders.in\

Difference between HTTP and HTTPS?

HTTP and HTTPS are both protocols for transfering data over the web, but HTTPS is the secure version of HTTP.

Example of HTTP:-

http://www.example.com.

Example of HTTPS:-

https://www.example.com.

HTML(Hyper Text Markup Language)?

HTML is the standard language used to create and design webpages it provides the structure for web content by using a series of elements ot tags.
