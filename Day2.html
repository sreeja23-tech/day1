HTML Tags

< !-- -->

This tag is used to apply comment in an HTML document.

< !DOCTYPE>

This tag is used to specify the version of HTML.

< a>

It creates a hyperlink or link.

< abbr>

Defines an abbreviation for a phrase or longer word.

< address>

Defines the author's contact information of the HTML article.

< applet>

Embeds a Java applet.(not supported by HTML5).

< area>

Defines the area of an image

< article>

Defines the self-contained content.

< aside>

Defines content aside from main content. Usually a sidebar.

< b>

Makes text bold.

< base>

Defines the base URL for relative URLs in the document.

< br>

Applies a line break.

< button>

Represents a clickable button.

< canvas>

Provides graphics space in a web document.

< col>

Defines a column in a table with shared properties.

< table>

Defines a table.

< u>

Underlined text.

< ul>

Unordered list
